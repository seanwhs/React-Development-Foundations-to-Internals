# React Development: Foundations to Internals

Welcome to **React Development: Foundations to Internals**, a comprehensive curriculum designed to bridge the gap between using React as a UI library and understanding it as a sophisticated execution engine. This repository takes you from the declarative foundations of component-based architecture to the complex mechanics of the Fiber reconciliation algorithm.

---

## 🗺️ Curriculum Map

| Module | Focus | Key Deliverable | Tooling |
| --- | --- | --- | --- |
| **1. Foundations** | JSX, Components, State, Props | Component Library | Vite, Vitest |
| **2. State** | Hooks & Lifecycle | Hook Customizer | React DevTools |
| **3. Internals** | Fiber & Reconciliation | Fiber Visualizer | React Source Code |
| **4. Performance** | Memoization, Concurrent React, RSC | Benchmarking Suite | Lighthouse, Profiler |
| **5. Synthesis** | Architecture | Production-grade App | Next.js, Concurrent Mode |

## 🎓 Instructor Implementation Notes

To ensure a high-quality learning experience, please adhere to the following guidelines:

* **Branching Strategy:** Encourage students to create a dedicated branch for each module (e.g., `feature/module-2-hooks`). This maintains a clean history and allows for easier code reviews.
* **Visualizing Internals:** Abstract concepts like the Fiber tree can be difficult to grasp. Utilize the `module-3-internals/diagrams/` directory to host SVG or Mermaid diagrams that map out the `FiberNode` structure.
* **Documentation:** Every module includes a `README.md` that references its specific place in the React Rendering Pipeline. Use these documents to connect theoretical concepts to actual code execution.
